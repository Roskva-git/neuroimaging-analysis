# fMRI preprocessing and analysis pipeline (SPM)

MATLAB script for preprocessing and statistical analysis of fMRI data using SPM12.

Course: PSY4320 Research Methods II (University of Oslo)

Pipeline includes:
- Slice timing correction
- Coregistration
- Segmentation
- Normalization
- Smoothing
- First-level contrast creation
- Second-level group analysis (one-sample t-tests)

Requirements:
- MATLAB
- SPM12

Note: File paths in the script must be changed to match the local data directory.
